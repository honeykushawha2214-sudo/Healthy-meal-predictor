# Healthy-meal-predictor.ipynb
This project uses Machine Learning to predict whether a meal is healthy or unhealthy using nutritional data.

 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

 Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest

 Features
- Data preprocessing
- Feature encoding
- Model training and testing
- Accuracy comparison
- Data visualization
- Feature importance analysis

## Dataset
Kaggle Healthy Eating Dataset
